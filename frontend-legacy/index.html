How does the search work?

This system uses two complementary AI techniques to find the most relevant alumni — not just those who share keywords with your query, but those who are semantically and relationally closest to what you're looking for.

Vector Search

Sentence-BERT + FAISS

Every alumni profile is converted into a 384-dimensional vector using a Sentence-BERT language model. This vector encodes the semantic meaning of the profile.

When you type a query, it is encoded into its own vector using the same model. FAISS then finds the profiles whose vectors are closest — a process called cosine similarity search.

Why this matters A search for "AI engineer" will surface profiles that mention "machine learning practitioner" or "deep learning researcher" — even without any exact keyword match.
Your query
→
SBERT encodes to vector
→
FAISS finds nearest neighbours
→
Top-N candidates ranked by cosine similarity

Graph Search

NetworkX Property Graph

All 500 alumni records are loaded into a property graph with 7 node types: Alumni, Company, Skill, Department, Batch, Location, and Mentor.

For each vector search candidate, a graph relevance score is computed using three signals: PageRank centrality, entity overlap, and relationship density.

Why this matters Two profiles may have identical vector scores, but the one who is also a mentor, worked at two companies you mentioned, and shares a batch with a reference profile will rank higher.
Vector candidates
→
PageRank + entity overlap + density
→
Graph score (0–1)

Hybrid Fusion — The Final Score

The two scores are combined using a configurable graph weight (α) that you can adjust with the slider in the search bar:

( 1 − α ) × Vector Score + α × Graph Score = Final Ranking Score

At α = 0.0, ranking is pure semantic similarity (vector only). At α = 1.0, ranking is purely graph-based (relationship richness). The default of α = 0.4 balances both signals. Each result card also shows a breakdown of its vector score, graph score, and final score so you can see exactly how the ranking was decided.
